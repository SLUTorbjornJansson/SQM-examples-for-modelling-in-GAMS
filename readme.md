# SQM in economic models in GAMS
A set of three different economic models implemented in GAMS with examples of tests and checks.

## Partial equilibrium
File: pe/multiMarket.gms
A partial equilibrium model with two commodities with synthetic price and quantity (data input, benchmarking, test runs, reporting code)
see pe/readme.md for furter information

## Computable General Equilibrium
File: cge/cge.gms
A CGE model with SAM input (synthetic), balancing, benchmarking, and reporting codem and test runs

see cge/readme.md for furter information

## A single farm model
File: farm/herd_dyn.gms
A single-farm optimization model depicting decision on a dairy herd and its feeding
